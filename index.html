Bubble Sort

Repeatedly compares adjacent elements and swaps them if they are in the wrong order. The largest elements "bubble up" to the end of the array with each pass.

Best: O(n)
Avg: O(n²)
Worst: O(n²)
Space: O(1)
Steps: 0 | Swaps: 0

// how bubble sort works

  • 1
    Start at the beginning of the array.
  • 2
    Compare the first two adjacent elements.
  • 3
    If the left element is greater than the right, swap them.
  • 4
    Move to the next pair and repeat until the end of the pass.
  • 5
    After each full pass, the largest unsorted element is in its correct position.
  • 6
    Repeat until no swaps occur — the array is sorted!
